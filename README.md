# 🛡️ JSI Command Center: Jewish Safety Index

**A Real-Time Strategic War Room for Tracking Jewish Safety across 50 US States and Global Jurisdictions.**

The JSI Command Center is a high-tech monitoring platform designed to provide instant visibility into risk levels for Jewish communities. By analyzing news sentiment and social media velocity, it provides a 0-100% "Danger Meter" to help users understand the current climate at a glance.

## 🚀 Key Features

- **War Room Interface:** High-contrast dark mode map with pulsing red pins.
- **Dynamic Risk Scores:** Real-time percentage indicators of local danger levels.
- **Dual-Theater View:** Toggle instantly between US Domestic (States) and International (Global) monitoring.
- **Community Intelligence:** A live chat feed for crowd-sourced reporting and verification.
- **Hyper-Local Geofencing:** Scores mapped to specific states using AI-driven entity extraction.

## 📊 Methodology

The JSI uses a weighted formula to calculate the **Danger Score**:
1. **Sentiment Analysis (40%):** Scanning for IHRA-defined antisemitic tropes.
2. **Mention Velocity (30%):** Tracking sudden spikes in hostile keywords.
3. **Verified Incidents (30%):** Fact-checking against local news and law enforcement reports.

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS, Leaflet.js (Map Engine)
- **Database:** Supabase (PostgreSQL)
- **Intelligence:** Python-based AI Sentiment Analyzer
- **Deployment:** Vercel / GitHub Pages
