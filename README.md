# 🛰️ JSI | War Room Command

### **Jewish Safety Index - Global Theater Monitor**

The **JSI War Room** is a real-time, geographic intelligence dashboard designed to monitor and visualize risk levels across the globe. Built for situational awareness and community safety, this tool combines automated data feeds with human intelligence (HUMINT) field reports.

---

## 🛠️ System Features

* **Global Risk Map:** Interactive dark-mode theater focused on high-risk geographic nodes.
* **Live Intelligence Ticker:** Scrolling real-time news feed pulling from global news nodes and internal JSI briefs.
* **Algorithmic Risk Scoring:** Locations are assigned a 0-100% risk factor based on current incident velocity and environmental data.
* **Field Report Transcription:** Allows users to manually log specific threats (Physical, Vandalism, Unrest) directly onto the theater map.
* **Emergency Signal (Panic Button):** One-tap mobile trigger to transmit a distress signal and location link to security contacts via SMS.

---

## 📖 How to Use the Command Center

### **1. Navigation & Search**
Use the **"Intel Search"** box in the sidebar to fly to any city or state. The system will automatically calculate a localized risk factor and drop a pulsing pin.

### **2. Transcribing a Field Report**
If you are witnessing an incident:
1. Search for the location.
2. Select the **Threat Type** from the dropdown menu.
3. Tap **"Post to War Room"**. A high-priority "!" pin will be logged on the map for all users to see.

### **3. Monitoring the Ticker**
The red ticker at the bottom provides a steady stream of breaking news. If the live connection is interrupted, the system fails over to pre-loaded JSI Intelligence briefs.

### **4. Mobile Installation (iPhone)**
To use JSI as a native app:
1. Open the Vercel link in Safari.
2. Tap the **Share** icon (square with arrow).
3. Select **"Add to Home Screen"**.
4. Launch the JSI icon from your home screen for a full-screen, distraction-free environment.
